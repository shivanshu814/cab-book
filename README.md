# Cab Booking System

This project is a cab booking system built using Next.js, a React framework. It allows users to book cabs for their desired destinations.

## Demo

You can watch a demo of the project https://drive.google.com/file/d/1M8Jwr5_izqI4ZPOCGDUlmZ7bI2IG5vT3/view?usp=sharing.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/shivanshu814/cab-book
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

   Open http://localhost:3000 to view the application in your browser.

## Features

- Users can enter their source and destination locations.
- Users can view the shortest path and distance between the source and destination.
- Users can book a cab for their journey by entering their email address.
- Email confirmation is sent to users after booking a cab.
- Users can cancel their booking.

## Technologies Used

- Next.js
- React
- Node.js
- Nodemailer
